# Python Resume & Document Parser

This project extracts structured data from resumes and documents (PDF/DOCX)
and outputs clean JSON for ATS or automation use cases.

## Features
- Resume parsing
- Text cleaning
- Structured JSON output
- Easy to extend

## Input
PDF / DOCX

## Output
JSON

## Use Cases
- ATS systems
- HR automation
- Document processing
